# CommandCore Constitution v1.0

## Locked Definition

CommandCore is an AI Enterprise Operating System.

It is not a chatbot, not only Jarvis, not only a productivity app, and not only an AI dashboard.

CommandCore is the operating layer for an AI company universe.

## Capability First Principle

We do not collect projects.

We collect capabilities.

Projects are reviewed, mined for reusable capabilities, scored, and either adopted, adapted, watched, or rejected.

## Agent Rule

CommandCore owns the architecture.
Agent engines are plugins.
Capabilities are reusable.
Companies consume capabilities.
